Path Replanning for Orientation-Constrained Needle Steering

This project implements a novel path-replanning algorithm for steerable needles in neurosurgery. It addresses challenges like tissue deformation, obstacles, and catheter control. The core algorithm combines 'Bubble Bending' for real-time path deformation with a 3D convex optimization method for path smoothing. This allows for closed-loop guidance using electromagnetic tracking, ensuring the needle reaches the target pose with high accuracy while avoiding sensitive structures.
